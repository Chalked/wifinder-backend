# WiFinder

- WiFinder is a mobile-optimized React application for locating quality free WiFi around downtown Denver.

### Technologies Used:

- CSS
- HTML
- JavaScript
- ReactJS
- KnexJS
- PostgreSQL
- ExpressJS
- Google Geocode API
- Heroku
- Firebase
- NodeJS
